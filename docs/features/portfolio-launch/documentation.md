# Portfolio Launch Documentation

## Key Decisions

- Portfolio code lives in a new dedicated repo rather than `demo-showcase`.
- The app uses Next.js because Vercel free is the primary host.
- All three app-style projects are framed as full-stack products.
- `Chocolate Craft House` is media-first in phase 1.
- `English4U` is the first live full-stack deployment target.
- `PetNest` is the second live full-stack candidate if hosting constraints remain reasonable.

## Notes

- Some ready images were copied from source projects where they already existed.
